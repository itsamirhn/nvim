# nvim

Personal Neovim config.

## Requirements

- [Neovim](https://neovim.io) >= 0.11
- [fzf](https://github.com/junegunn/fzf)
- [ripgrep](https://github.com/BurntSushi/ripgrep)
- [rust-analyzer](https://rust-analyzer.github.io)
- [pyright](https://github.com/microsoft/pyright)
- A [Nerd Font](https://www.nerdfonts.com/) set in your terminal (for file icons). I use [Hack Nerd Font](https://www.nerdfonts.com/font-downloads).

Clone to `~/.config/nvim` and run `nvim`. [lazy.nvim](https://lazy.folke.io) bootstraps itself on first launch.
